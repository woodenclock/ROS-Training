## Introduction to ROS2 Basics

This repository contains hands-on exercises (Practice 1 to 10) aimed at building foundational knowledge and practical skills in ROS 2 development. These exercises support the four-day **ROS 2 Basics Training** conducted by  **ROS-Industrial Asia Pacific** and are intended to reinforce key concepts through hands-on learning.

The exercises are also specifically designed for use with TurtleBot3, a physical mobile robot platform widely adopted in ROS 2 education and experimentation.

## Prerequisites

Practical exercises are conducted using the following virtual machine environment:

- **Ubuntu 22.04**
- **ROS 2 Humble**

To set up your virtual machine for the **ROS 2 Basics Training**, please follow the instructions in [this setup guide](./pc_setup/README.md).
